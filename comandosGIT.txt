CRIAR NOVO PROJETO (NECESSARIO CRIAR REPO EM www.github.com)

git init - cria um repositorio na diretoria

git add .

git status

git commit -m "MENSAGEM"

git branch -M master

git remote add origin git@github.com:pfsfreitas/aulagithub.git

git push -u origin main




ENVIAR NOVA VERSÃO PARA SERVER:

git status

git add .

git commit -m "MENSAGEM"

git push




CLONAR PROJETO EXISTENTE NO GitHub

git clone git@github.com:pfsfreitas/aulagithub.git




REMOVER ARQUIVOS DA AREA DE STAGE:

git status 

git reset




DESFAZER MODIFICAÇÕES NÃO SALVAS

git status

git reset 

git clean -df

git checkout -- .




DESFAZER ULTIMO COMMIT SEM DESFAZER MODIFICAÇÕES NOS ARQUIVOS:

git status

git reset --soft HEAD~1




APAGAR COMMITS E ALTERAÇÕES POSTERIORES A ESSE COMMIT

git status

git reset --hard <codigo do commit>
